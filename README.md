<h1>Sound-Game-Jam-2025-Navigation-of-Soltrianius</h1>

<h2>Game Summary: </h2>
In this game that was developed for the Sound Game Jam 2025, it is inspired by Catan, you are the navigator of the Celestial Cosmos Galaxy Exploration Team who is on a mission to discover the Soltrianius Galaxy, a cozy uninhabited galaxy that brings hope to many with its vast planets. You and your team hope to find a place for their civilization to be able to move to as their home planet is in danger.

itch.io Page for More Information on Navigation of Soltrianius: https://kh74.itch.io/navigation-of-soltrianius

<h2>Features</h2> 
This repository includes:
- The interactive User Interface simulation of Navigation of Soltrianius
- The foundational Object-Oriented Programming design of the in-game elements

<h2>Tech Stack</h2>
The following technology was used to create Navigation of Soltrianius: C#, WinForms, Visual Studio (VS) Code, itch.io, GitHub, AutoDesk Sketchbook, Canva, and Figma.

<h2>Design Patterns</h2>
The following design patterns were used during development: Lean Methodologies, Scalable Design, Agile Development, Object Oriented Programming (OOP), and Model View Controller (MVC) Design.

<h2>Credits</h2>
